# v0.1.52 — Skills Marketplace (claude-plugins.dev)

**Date:** 2026-03-10

## Features

- `RegistryService` with `RegistrySource` Protocol — multi-source abstraction for skill registries
- `ClaudePluginsDevSource` — hits `https://claude-plugins.dev/api/skills`, 52k+ community skills
- `RegistryPage`, `RegistrySkill`, `RegistrySkillMeta` Pydantic v2 models
- Memory cache 300s per `(source, query, offset)`
- `registry` lazy property on `Client`
- Marketplace tab in Skill Studio: search, pagination, install button, `is_installed` check
- Install downloads `SKILL.md` via `metadata.rawFileUrl` (direct GitHub raw link)
