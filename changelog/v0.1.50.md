# v0.1.50 — Model selector from provider /models API

**Date:** 2026-03-10

## Features

- Settings → LLM Provider now shows `st.selectbox` populated from provider `/models` endpoint
- `_ModelInfo(BaseModel)` with `id`, `name`, `label` property
- `_fetch_models()` cached in `st.session_state` per `(mode, key[:8])` — no redundant API calls
- Falls back to `st.text_input` when no API key is configured
- Pre-selects current model or default model; inserts at top if not in returned list
