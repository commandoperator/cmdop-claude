# v0.1.64 — Changelog system

**Date:** 2026-03-10

## Features
- `ChangelogService` — parses `changelog/vX.Y.Z.md` files (list, get, get_latest, write_entry)
- MCP tools `changelog_list` and `changelog_get` — browse release history from Claude Code
- Streamlit "Changelog" tab — version selector, full markdown entry, current version highlighted
- `inject-tasks` hook — prepends version banner (`📦 cmdop-claude vX.Y.Z | date | title`) to every prompt context

## Improvements
- Bundled `/commit` SKILL.md — canonical changelog format enforced (title + `**Date:**` mandatory), git tags instead of version branches (PyPI convention)
- `Client.changelog` lazy property — `ChangelogService` accessible from Python API
- README refreshed — Skill Studio, Changelog system, updated tab count (12), architecture section

## Bug Fixes
